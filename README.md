Light On/Off Toggle

This project demonstrates a basic use of the JavaScript onclick function to control a light bulb image. By clicking a button, the bulb switches between ON and OFF states, showing how event handling and DOM manipulation work in a simple, interactive way.

✨ Features

Toggle light bulb image with a click ,
Beginner-friendly JavaScript example ,
Easy to customize for other interactive elements .

🚀 Purpose

A fun and simple project for learning event handling in JavaScript. Perfect for beginners exploring how user actions can trigger changes on a webpage.
